# Placement Portal (MERN + Redux Toolkit)

A full-stack Placement Management System built using **MERN Stack**.  
It connects **Students** looking for job opportunities with **Recruiters** posting openings.

---

## Features

### ✅ Student 
- User Authentication (Signup / Login)
- View all available job listings
- Apply for jobs
- Update personal profile:
  - Add / remove skills
  - Upload resume
  - Update basic information (name, education, etc.)
- Track applied jobs

### ✅ Recruiter  
- User Authentication (Signup / Login)
- Add & manage companies
- Post jobs inside created companies
- View applicants for posted jobs
- View applicant resumes
- Accept or reject candidates

---

##  Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React.js, Redux Toolkit, Tailwind  |
| Backend | Node.js, Express.js |
| Database | MongoDB |
| Authentication | JWT (JSON Web Tokens) |
| State Management | Redux Toolkit |
| Storage | Multer / Cloud storage for resume uploads |
